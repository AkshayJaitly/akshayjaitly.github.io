About

Engineering where data meets ML

I'm a Senior Software Engineer at Capital One, building and deploying machine-learning and LLM systems that power data-driven decisions in fintech. Day to day that means RAG models, AI agents, Databricks & Iceberg data lakes, and MLOps on SageMaker.

Before this chapter I spent two and a half years at Siemens Digital Industries Software building Go REST APIs and Angular apps for smart, sustainable buildings — including a semantic-enrichment engine for building data and IoT connectors handling high throughput in real time.

Earlier at Capital One I wrote the Go + AWS Lambda rewards API behind Shop with Rewards, and built big-data pipelines in Spark & Scala. I got my start as a HackerOne security researcher and a published undergrad researcher in computer vision. I like clean abstractions, observable systems, and code that holds up under real load.
